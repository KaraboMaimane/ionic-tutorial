How to install Ionic CLI and Cordova

I'll carefully take you through the process of the installing the application according to my knowlege


Before we even get started we will require the following:

  • Node.js(the latest version)
  • A stable internet connection
  • A brain

Step 1: Download and install Node.js

Step 2: Open a terminal on your computer it can be command prompt or Git/Powershell(whatever you prefer)

Step 3: In your terminal we will install the Ionic CLI(Command Line Interface) as well as Cordova and to achieve this you will require to enter this in your terminal

npm install -g ionic cordova

A little sidenote: npm means that we are using the Node Package Manager
This little -g we are adding means that it will be installed gloablly

Step 4: Once the downloading of all dependencies are completed guess what??? We are done!!!
