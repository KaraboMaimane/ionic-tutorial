Creating A New Ionic Project

I will briefly explain how to create a new project in Ionic


Step 1: Firstly we are going to open our terminal of choice like we did in the previous method when installing the Ionic CLI(If you havent done this then i suggest that you do so now before proceeding). Link to instaling the CLI

Step 2: We are then going to enter the following command in our terminal:

ionic start [ProjectName] [ProjectType]
ionic start helloWorld blank

A little sidenote: There are various project types that you can choose from to achieve this then you are going to leave out "blank" and type this in your command:

ionic start helloWorld

Step 3: Once the terminal has completed installing the dependencies then you are as good as done... Congratulations! you have created your first ionic application!!!
